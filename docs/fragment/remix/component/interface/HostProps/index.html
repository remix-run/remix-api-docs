HostProps

View Source

Summary

Shared host-element props accepted by all built-in DOM element types.

Signature

interface HostProps<eventTarget> {
  children?: RemixNode;
  innerHTML?: string;
  key?: any;
  mix?: MixValue<eventTarget>;
}

Properties

children

Child nodes to render inside the element.

innerHTML

Set the innerHTML of the element directly. When provided, children are ignored. Use with caution as this can expose XSS vulnerabilities if the content is not sanitized.

key

The reconciliation key for the element.

mix

Mixins to apply to the element.
