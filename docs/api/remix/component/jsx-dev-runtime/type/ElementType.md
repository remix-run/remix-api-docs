# ElementType

<a href="https://github.com/remix-run/remix/blob/remix@3.0.0-alpha.2/packages/component/src/lib/jsx.ts#L9" target="_blank">View Source</a>

## Summary

Any valid element type accepted by JSX or `createElement`.
- `string` for host elements (e.g., 'div')
- `Function` for user components

## Signature

```ts
type ElementType = string | Function;

```
