# jsxDEV

## Summary

Creates a Remix virtual element.

## Signature

```ts
function jsx(type: ElementType, props: ElementProps, key: string): RemixElement;

```

## Parameters

### `type`

Host tag or component function.

### `props`

Element props.

### `key`

Optional reconciliation key.

## Returns

A Remix virtual element.
