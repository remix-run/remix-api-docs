# run

## Summary

Starts the client-side Remix component runtime for the current document.

## Signature

```ts
function run(init: RunInit): AppRuntime;

```

## Parameters

### `init`

Runtime hooks for loading modules and resolving frames.

## Returns

The running application runtime.
