# createCookie

## Summary

Creates a new cookie object.

## Signature

```ts
function createCookie(name: string, options: CookieOptions): Cookie;

```

## Params

### `name`

The name of the cookie

### `options`

Options for the cookie

## Returns

A new [`Cookie`](/api/remix/cookie/class/Cookie/) object
