# equal

## Summary

Returns true if matches `a` and `b` have equal specificity.

## Signature

```ts
function equal(a: Match, b: Match): boolean

```

## Parameters

### `a`

the first match to compare

### `b`

the second match to compare

## Returns

true if `a` and `b` have equal specificity
