# lessThan

## Summary

Returns true if match `a` is less specific than match `b`.

## Signature

```ts
function lessThan(a: Match, b: Match): boolean;

```

## Parameters

### `a`

the first match to compare

### `b`

the second match to compare

## Returns

true if `a` is less specific than `b`
