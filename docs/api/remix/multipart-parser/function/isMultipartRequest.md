# isMultipartRequest

## Summary

Returns true if the given request contains multipart data.

## Signature

```ts
function isMultipartRequest(request: Request): boolean;

```

## Parameters

### `request`

The `Request` object to check

## Returns

`true` if the request is a multipart request, `false` otherwise
