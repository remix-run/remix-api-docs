ParseOptions

View Source

Summary

Options passed to parse and parseSafe.

This mirrors ValidationOptions, but also supports a convenience abortEarly option at the top level.

Signature

type ParseOptions = StandardSchemaV1.Options & {
  abortEarly?: boolean;
  errorMap?: ErrorMap;
  locale?: string;
};
